# BKSouls

BKSouls is a Unity action RPG prototype that blends Souls-like combat with roguelike dungeon runs and shelter progression.

Create a character, choose a starting class, enter dangerous dungeon spaces, collect resources, and return to your shelter to recover, upgrade, and prepare for the next run.

## Features

- Souls-inspired third-person melee combat
- Character creation with class, body type, hair, color, and name options
- Multiple starting classes with different stats and equipment
- Roguelike dungeon flow with room progression, clear/fail results, and run rewards
- Shelter hub progression with building, upgrades, and resource management
- Grid-based inventory, equipment slots, shops, forge, item selling, and storage
- Persistent character save data across runs
- Unity Netcode-based architecture for networked gameplay experiments

## Gameplay Loop

1. Create or load a character.
2. Prepare equipment and upgrades in the shelter.
3. Enter a dungeon run.
4. Fight enemies, spend runes, loot items, and progress through rooms.
5. Return to the shelter with rewards or recover after failure.
6. Upgrade your shelter and character systems before the next run.

## Tech Stack

- Unity
- Netcode for GameObjects
- Unity Input System
- Unity AI Navigation
- Cinemachine
- TextMeshPro / UGUI

## Current Status

BKSouls is in active prototype development. Core systems such as character creation, combat, inventory, dungeon progression, save data, and shelter progression are implemented, but balance, content, polish, and platform support are still evolving.

## Notes

This repository contains a work-in-progress Unity project. Some systems may require specific scenes, prefabs, or editor setup to run correctly.
